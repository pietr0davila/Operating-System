# Boot básico
Quando a placa mãe recebe energia e inicializa, ela carrega a BIOS (Basic Input Output System)
# MBR (Master boot record)
Setor 0 do disco
512 Bytes
Endereço lógico do disco é LBA 0 - endereço físico 0
O MBR começa no setor 0mas é copiado para 0x7C00 - Antes disso vem a IVT (Interrupting vector table) e a própria BIOS, que não pode ser sobreescrita  
- O MBR contém o código do bootloader primário (446 Bytes)
- Tabela de partições
- Assinatura de boot - 0x55AA (últimos 2 bytes)
